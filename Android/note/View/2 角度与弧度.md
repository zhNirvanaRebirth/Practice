# 角度与弧度  
当我们在操作画布的旋转以及进行一些三角函数的计算时，就会涉及到角度或弧度  
## 进制  
* 角度：60进制  
* 弧度：10进制  
## 定义  
* 角度：从圆心向圆周做两条射线，形成一个夹角和夹角正对的一段弧，当弧长是圆周长的360分之一时，这个夹角为1度  
* 弧度：从圆心向圆周做两条射线，形成一个夹角和夹角正对的一段弧，当弧长等于圆半径时，这个夹角为1弧度  
## 转换关系  
```  
360(角度) = 2π(弧度)  
知道角度为x，求其对应的弧度y：y = (x/360) * 2π;  
知道弧度为x，求其对应的角度y：y = (x/2π) * 360;  
```  
## 角度与时针  
顺时针方向，角度增大
