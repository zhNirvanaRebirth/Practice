## 坐标系  
### 坐标原点  
默认的坐标原点：屏幕的左上角  
### 坐标轴  
屏幕上，自左向右为x轴正方向，自上向下为y轴正方向  
### View中与坐标轴有关的函数  
```  
getTop();       //获取当前view左上角到父view顶部的距离  
getLeft();      //获取当前view左上角到父view左侧的距离  
getRight();     //获取当前view右下角到父view右侧的距离  
getBottom();    //获取当前view右下角到父view下部的距离  
```  
**这些方法都是获取到的都是相对于父view的距离，单位：像素**  
### MotionEvent中与坐标轴相关的函数  
```  
getX();         //触摸点相对于当前view中的坐标系x坐标位置（当前view的坐标，默认坐标原点在view的左上角，当然也可以通过程序设置）  
getY();         //触摸点相对于当前view中的坐标系y坐标位置  
  
getRawX();      //触摸点相对于屏幕默认坐标系x坐标的位置  
getRawY();      //触摸点相对于屏幕默认坐标系y坐标的位置  
```
