# 计算机概论  
## 什么是计算机  
计算机实际上就是接收用户输入的数据与资料，经过中央处理器的数字与逻辑处理单元的计算与处理后，产生或存储有用的资料的机器

### 五大单元  
* 输入单元  
* 输入单元  
* CPU内部的控制单元  
* 逻辑计算处理单元  
* 内存  
## CPU  
CPU（Central Processing Unit）：中央处理器，是主机部分的核心组件；*cpu是具有特定功能的晶片，里面含有各种微指令集*，因此，如果我们想让主机进行某些操作时，必须在cpu上具有相应的微处理指令集才行；cpu包含两个主要的单元：*逻辑运算与处理单元和控制单元*，逻辑运算与处理单元主要负责程式的运算与逻辑判断，控制单元主要协调各周边元件与各单元间的工作  
cpu的输入来自内存，而经过cpu处理好的资料也需要经过内存再到输出单元  
### CPU的架构  
#### 精简指令集（Reduced Instruction Set Computer，RISC）  
单个指令的执行时间很短，完成的动作也很简单，执行效能很高，但是处理事情很复杂时，则需要多条指令  
目前使用RISC架构的CPU：  
* Oracle的SPARC系列  
* IBM的Power Architecture系列  
* ARM的arm系列  
#### 复杂指令集（Complex Instruction Set Computer，CISC）  
单个指令集执行的时间较长，但是每条指令集可以处理的工作比较丰富，常见使用复杂指令集的CPU：AMD，Inter，via等的x86架构的cpu  
  
*位元，cpu一次能读取的最大数据，现在常见的是32bit和64bit*  
> 不同的x86架构的cpu的差异：整体结构（第二层快取和每次运作可执行的指令数等）、微指令集的不同（这个指令集可以加速多媒体程式的运作、加强虚拟化的效能、增加能源效率，降低cpu的耗电量等）  
##### Intel/AMD的x86架构中的一些微指令集  
* 多媒体微指令集：MMX，SSE，SSE2，SSE3，SSE4，AMD-3DNow!  
* 虚拟化微指令集：Intel-VT，AMD-SVM  
* 省点功能：Intel-SpeedStep，AMD-PowerNow! 
* 64/32位元相容技术：AMD-AMD64，Intel-EM64T  
#### 外频与倍频  
* 外频：cpu与外部元件进行资料传输的速度  
* 倍频：cpu内部用来加速工作效能的一个倍数  
* 外频与倍频的乘积才是cpu的时脉  
*超频：就是修改cpu的外频或倍频而获得一个更高的频率，但是一个倍频值在厂商生产CPU的时候就设定了，因此一般只能修改cpu的外频*  
> 双核cpu：在cpu中具有两个运算单元  
**综上：衡量一个cpu的好坏（快慢）的因素有：  
* 时脉，就是cpu每秒可以执行多上次运算  
* cpu上的微处理集，微处理集的不同，各种性能也不同  
* cpu的架构，不同的架构使得cpu的指令处理速率与时间不同，所以就算拥有同样时脉，架构不同，执行效率也不同

## 电脑上的计算单位  
bit Byte Kilo Mega Giga Tera Peta Exa Zetta  
> 换算关系：1Byte = 8bit 1K = 1024Byte（2进制）  
>   
> cpu运算速率通常使用的是MHz和GHz，但在网络上使用的单位是Mbps（Mbits per second）  
*一般来说，电脑上的存储文档什么的使用的是2进制，但是比如cpu的运算速率使用的是10进制（执行速率是1GHz的意思就是1000*1000*1000Hz，每秒执行这么多次，所以电脑的运算能力还与这个执行速度有关，也就是所说的主频）*  
## 与个人电脑相关的设备元件  
> 晶片组：联通各部元件，晶片组设计的优劣会很大程度上影响效能  
### 早期的晶片组  
* 北桥：负责连接速度较快的cpu、内存以及显卡  
* 南桥：负责连接速度较慢的装置，包括硬盘、usb、网卡等等  
### 新的cpu设计  
Intel使用QPI(Quick Path Interconnect)和DMI技术，AMD使用Hyper Transport技术，使cpu直接与内存、显卡进行联通  
**新的cpu设计：将内存控制器整合到cpu内部，使cpu和内存直接**  
### 内存的时脉  
内存的时脉是由cpu中的内存控制器决定的，而内存的时脉和cpu的位元决定了cpu每秒可以从内存中读取的数据  
> 如64位元的cpu，内存的时脉是1600MHz，则CPU从内存中读取数据计算公式：1600MHz*64bit = 1600MHz*8byte = 12.8Gbyte/s  
*cpu的时脉表示的是cpu的运算速率*
