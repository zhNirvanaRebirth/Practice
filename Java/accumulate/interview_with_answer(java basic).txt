10. 重载和重写的区别
	重载：有相同的返回值和方法名，参数的类型或个数不一样（或参数类型和个数都不一样）
	重写：在子类重写父类的方法，返回值，方法名和参数类型与个数都相同，但实现不同
	
11. Java 面向对象编程三大特性: 封装 继承 多态
	封装
	继承
	多态
		封装：java中所有类，类中的属性，方法都有访问修饰符
		继承：java中的类可以继承父类的属性与方法（非私有，单继承）
		多态：一个子类的实例对象可以用父类去引用
		
12. String StringBuffer 和 StringBuilder 的区别是什么 String 为什么是不可变的
	String：大小固定
	StringBuffer：大小可变，线程安全
	StringBuilder：大小可变，非线程安全
	String对象所指向的是字符数组的地址，而这个字符数组的大小是确定了的，当我们修改String的值时，实际上会新申请内存创建新的数组

17. 接口和抽象类的区别是什么
	接口：不能被实例化，只能被实现，不能有已实现的方法
	抽象类：不能被实例化，只能被继承，可以有部分已实现的方法
	
18. 成员变量与局部变量的区别有那些
	成员变量：作用范围为整个类，共有成员变量可以被实例化对象访问与修改
	局部变量：作用范围为所在代码块，不能被外部访问

20. 什么是方法的返回值?返回值在类的方法里的作用是什么?
	执行一个方法，方法执行完毕后的告诉调用者的值
	方法的返回值可以告诉调用者方法的执行情况与结果
	
21. 一个类的构造方法的作用是什么 若一个类没有声明构造方法,该程序能正确执行吗 ?为什么?
	类的构造方法用于实例化一个类的对象，做响应的初始化等工作（解释类实例化的过程，也就是new关键字）、
	一个类没有声明构造方法，程序可以正常执行，即可以正常实例化类的对象，因为类当没有声明类的构造方法的时候，在编译的时候，编译器会为类加上一个没有参数的默认构造方法
